Arithmetic Command Shell v1.0
========================================
Welcome to the Web-Based Arithmetic Shell!
Type 'help' for available commands.
Example commands to try:
add 5 3 → Adds two numbers
sub 10 4 → Subtracts second number from first
mul 6 7 → Multiplies two numbers
div 20 5 → Divides first number by second
clear → Clears the terminal
help → Shows this help message
